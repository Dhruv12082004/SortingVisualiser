#  Sorting Visualizer

An interactive **Sorting Algorithm Visualizer** built using **HTML, CSS, and JavaScript**.  
It visually demonstrates how various sorting algorithms work by animating comparisons and swaps in real time.

---

##  Features
- Implements major sorting algorithms:
  - **Bubble Sort**
  - **Selection Sort**
  - **Insertion Sort**
  - **Quick Sort**
- Adjustable **array size** and **sorting speed**  
- Color-coded bars to represent comparisons and swaps  
- Smooth animations using JavaScript and CSS transitions  
- Simple and responsive UI for desktop browsers
